**Machine Learning (Aprendizado de Máquina)** é uma área da Inteligência Artificial em que computadores aprendem padrões a partir de **dados**, em vez de serem programados com regras fixas para cada situação. 🤖📊

Em termos simples:
➡️ Você mostra **muitos exemplos** para o computador.
➡️ Ele encontra **padrões** nesses dados.
➡️ Depois consegue **fazer previsões ou decisões** sozinho com novos dados.

---

## Como funciona (ideia básica)

1. **Coleta de dados** – por exemplo: fotos, textos, números.
2. **Treinamento do modelo** – um algoritmo analisa os dados e aprende padrões.
3. **Teste / previsão** – o modelo usa o que aprendeu para prever resultados.

Exemplo simples:

* Você mostra **10.000 fotos de gatos e cachorros**.
* O modelo aprende diferenças entre eles.
* Depois ele consegue identificar se uma **nova foto** é gato ou cachorro. 🐱🐶

---

## Principais tipos de Machine Learning

### 1️⃣ Aprendizado supervisionado

O modelo aprende com **dados já rotulados**.

Exemplo:

* Prever preço de casas
* Identificar spam em emails

---

### 2️⃣ Aprendizado não supervisionado

Os dados **não têm rótulos**, e o algoritmo encontra padrões sozinho.

Exemplo:

* Agrupar clientes com comportamentos parecidos.

---

### 3️⃣ Aprendizado por reforço

O sistema aprende **tentando e recebendo recompensas ou punições**.

Exemplo:

* IA aprendendo a jogar jogos ou controlar robôs. 🎮

---

## Exemplos do dia a dia

Você usa machine learning sem perceber:

* Recomendação de filmes na Netflix 🎬
* Sugestões de vídeos no YouTube
* Reconhecimento facial no celular 📱
* Filtros de spam no email
* Assistentes virtuais como Google Assistant
